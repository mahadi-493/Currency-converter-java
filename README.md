Currency Converter (Java)<br>

Project Overview<br>
Currency Converter is a Java-based application that allows users to convert one currency into another.<br>
This project is developed as part of a Java / OOP laboratory to practice core Java concepts and basic application logic.<br>

The application demonstrates clean code structure, user input handling, and simple mathematical operations.<br>

--------------------------------------------------

Features<br>
- Convert currency from one unit to another<br>
- Supports multiple currencies<br>
- Simple and user-friendly interface<br>
- Accurate conversion logic<br>
- Easy to extend with more currencies<br>

--------------------------------------------------

Concepts Used<br>
- Java Programming<br>
- Object-Oriented Programming (OOP)<br>
- Conditional statements<br>
- User input handling<br>
- Basic arithmetic operations<br>

How to Run the Project<br>

Prerequisites<br>
- Java JDK 8 or higher<br>
- Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans) or Terminal<br>

Run Using Terminal<br>

javac CurrencyConverter.java<br>
java CurrencyConverter<br>

Run Using IDE<br>
1. Open the project in your IDE<br>
2. Locate the main Java file<br>
3. Run the program<br>

Example Use Case<br>
